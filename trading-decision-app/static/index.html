把多智能体投研 × 策略库装进一张工作台

传统的"AI 选股"通常是单次 LLM 问答 — 给一段提示，得一段回答。本系统不一样： 它把 TradingAgents 的 12 个智能体（4 个分析师、2 个研究员、3 个风险分析师、3 个经理 + 交易员） 组合成一条完整的投研流水线，**实时**显示每个 agent 的进度、工具调用、辩论过程， 最终给出 5 档评级（Buy / Overweight / Hold / Underweight / Sell）— 并自动从 78 条策略库挑出 Top-5 可执行方案。

怎么用

  1. 去 "AI 智能决策" Tab — 输入股票代码、日期、LLM、研究深度等参数。
  2. 点击"启动"，每次启动都会开一个新窗口；多个窗口可以同时运行不同标的。
  3. 实时观察左侧侧边栏切换不同章节：分析师报告 / 辩论 / 风险讨论 / 最终决策。
  4. 完成后保存到历史记录；点击左侧"📜 历史决策"可以随时回看、下载报告。

风险提示与免责声明

① 通用风险

  • 所有交易策略都无法保证盈利。历史回测和理论模型不代表未来表现。
  • 市场可能在任意时刻发生与你预期相反的剧烈变动（黑天鹅、政策、流动性枯竭）。
  • "低风险"策略只是相对概念，极端环境下可能出现远超历史的回撤。

② AI 决策的局限

  • 多智能体辩论可以降低单一视角偏差，但不能消除模型的训练数据偏差。
  • LLM 可能生成看似合理但事实错误的"幻觉"，请将其输出作为参考而非结论。
  • 日期 / 行情 / 财报数据来自公开 API，可能滞后或缺失。

③ 杠杆与衍生品风险

  • 期货、永续合约、保证金交易可能损失超过本金，存在爆仓与穿仓风险。
  • 期权买方可能亏损全部权利金；期权卖方在裸卖时面临理论上无限亏损。

④ 心理与执行风险

  • 策略本身是中性的，真正的风险在执行人身上：不止损、加杠杆、报复性交易。
  • 同一个策略不同人执行，结果可能南辕北辙。请先用小仓位验证规则与心理。
